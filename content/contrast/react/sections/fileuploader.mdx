# React Bootstrap File Uploader

The React Bootstrap FileUploader allows you to upload files using the drag and drop functionality.

## Importing the React Bootstrap FileUploader Component

To use the Contrast React Bootstrap DatePicker component, you import `CDBFileUploader` into your project.

```js
import { CDBFileUploader } from "cdbreact";
```

# Getting Started

Get started with `cdbreact` - An elegant PRO UI Kit and PRO reusable components for building mobile-first, responsive websites and web apps

It will take less than 5 minutes.

Step 1: Create a React App using create-react-app

`npx create-react-app <appname>`

Step 2: Navigate into the project's folder

Step 3: Install `cdbreact`

`yarn add cdbreact`

Step 4: Import `cdbreact` in your project

```js
import React from 'react';
import { CDBAlert } from 'cdbreact';

const Alert = () => {
  return <CDBAlert color="primary">A simple alert built with contrast design</CDBAlert>;
};
export default Alert;
```

Step 5: Start coding! Yes, its that easy.

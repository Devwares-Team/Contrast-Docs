# React Bootstrap ButtonGroup

A button group in React Bootstrap wraps a bunch of buttons into a single line or stack in a vertical column. The Button Group component generates a group to which buttons can be added. Of course, a button group can only have one button selected at any given moment. As a result, selecting a button in a button group deselects all of the other buttons in the group.

The visual look of the buttons on the screen has nothing to do with button groupings. Instead, button groups merely organize the buttons into logical groups.
The Button Group component controls whether a group of buttons is chosen or not.

Applications of React Bootstrap button group:

We can see the application of these component on our:

* Group of pricing options
* Group of licenses you can purchase on our website Pro page


## Importing the Contrast React Bootstrap ButtonGroup component

Start by importing 'CDBBtnGrp' into your project to use the Contrast React Bootstrap ButtonGroup component.

```javascript
import { CDBBtnGrp } from "cdbreact";
```

## Multicolored Button Groups

Alongside the `CDBBtnGrp` component, we imported the `CDBBtn` component to create our buttons.

The `color` prop also affects the Contrast React Bootstrap ButtonGroup, as it gives the buttons in the ButtonGroup a background color, depending on the values passed into the prop.

  <div class="container">
    <button type="button" class="btn btn-primary">Click me</button>
    <button type="button" class="btn btn-secondary">Click me</button>
    <button type="button" class="btn btn-success">Click me</button>
    <button type="button" class="btn btn-danger">Click me</button>
    <button type="button" class="btn btn-dark">Click me</button>
  <button type="button" class="btn btn-warning">Click me</button>
  </div>

```javascript
import React from "react";
import { CDBBtnGrp, CDBBtn, CDBContainer } from "cdbreact";

const ButtonGroup = () => {
  return (
    <CDBContainer>
          <CDBBtnGrp size="lg">
            <CDBBtn color="primary">click me</CDBBtn>
            <CDBBtn color="secondary">click me</CDBBtn>
            <CDBBtn color="success">click me</CDBBtn>
            <CDBBtn color="danger">click me</CDBBtn>
            <CDBBtn color="dark">click me</CDBBtn>
            <CDBBtn color="warning">click me</CDBBtn>
          </CDBBtnGrp>
    </CDBContainer>
  );
  export default ButtonGroup;
};
```

## Button Group with Dropdown

In this tutorial we also use the [CDBIcon](https://www.devwares.com/docs/contrast/react/components/icon) for our icons, [CDBBtn](https://www.devwares.com/docs/contrast/react/components/buttons) to create buttons, and the [CDBDropdown](https://www.devwares.com/docs/contrast/react/components/dropdown) component to create our dropdowns alongside the `CDBButtonGroup` component. 

  <div class="container d-flex">
    <button type="button" class="btn m-0 btn-md btn-primary">Click me</button>
    <button type="button" class="btn m-0 btn-md btn-secondary">Click me</button>
    <button type="button" class="btn m-0 btn-md btn-success">Click me</button>
    <div class="dropdown">
      <button
        class="btn m-0 btn-warning text-dark dropdown-toggle"
        type="button"
        id="dropdownMenuButton"
        data-toggle="dropdown"
        aria-expanded="false"
      >
        Dropdown
      </button>
      <ul class="dropdown-menu" aria-labelledby="dropdownMenuButton">
        <li><a class="dropdown-item" href="#">Action</a></li>
        <li><a class="dropdown-item" href="#">Another action</a></li>
        <li><a class="dropdown-item" href="#">Something else here</a></li>
      </ul>
    </div>
  </div>

```javascript
import React from "react";
import { CDBBtnGrp, CDBIcon, CDBBtn, CDBDropDown, CDBDropDownToggle, CDBDropDownMenu, CDBDropDownItem, CDBContainer } from "cdbreact";

const ButtonGroup = () => {
  return (
    <CDBContainer>
          <CDBBtnGrp>
            <CDBBtn color="primary">click me</CDBBtn>
            <CDBBtn color="secondary">click me</CDBBtn>
            <CDBBtn color="success">click me</CDBBtn>
          </CDBBtnGrp>
          <CDBBtnGrp>
            <CDBDropDown>
              <CDBDropDownToggle color="warning">Dropdown<Icon fas icon="caret-down"/></CDBDropDownToggle>
              <CDBDropDownMenu dropright>
                <CDBDropDownItem>Action 1</CDBDropDownItem>
                <CDBDropDownItem>Action 2</CDBDropDownItem>
              </CDBDropDownMenu>
            </CDBDropDown>
          </CDBBtnGrp>
    </CDBContainer>
  );
  export default ButtonGroup;
};
```

# Contrast React Bootstrap ButtonGroup Props

## API Reference: Contrast ButtonGroup Properties

The table below shows the configuration options of the `CDBBtnGrp` component.

| Name            | Type        | Default      |   Description| Example      |
| :------------- | :----------: | -----------: | :----------: | -----------: |
| className      | String       |              |Adds custom classes	      |     <CDBBtnGrp className="myClass" ... />  |
| role           | String       | group        | Change default component's role | <CDBBtnGrp role="group" ... />  |
| size           | String       | medium       |  	Determines buttons size, available values: `[sm, lg]`. | <CDBBtnGrp size="lg" ... />  |
| vertical       | Boolean      | false        | Changes group orientation to vertical | <CDBBtnGrp vertical .../>  |

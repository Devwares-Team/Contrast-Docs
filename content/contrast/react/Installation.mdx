Step by step guide on how to install `cdbreact`

## Prerequisites:

To use CDBReact in your project, you have to install React. CDBReact works with Node versions 6 or higher.

### Install CDB React

To install cdbreact, go to your terminal or command prompt and run the command:

> NPM

`npm install --save cdbreact`

> Yarn

`yarn add cdbreact`

> **_To run Your App:_**

> NPM

`npm run start`

> Yarn

`yarn start`
